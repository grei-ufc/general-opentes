## 📌 Descrição da Atividade

> Nesta semana foi dada continuidade ao estudo do modelo de recursos elétricos distribuídos da EPRI, o OpenDER.
> São fornecidos um arquivo com a descrição das características do modelo, com diagramas e equações que descrevem seu comportamento,
> além de um repositório no github com exemplos implementados utilizando o modelo. Com isso, na semana anterior foi realizada a leitura dos pontos
> mais relevantes para o entendimento da ferramenta. O restante das informações presentes no documento foram interpretadas como material para consulta
> posterior, dadas as informações presentes. Desta maneira, foi iniciado o estudo dos códigos fornecidos pelos desenvolvedores como exemplos,
> entretanto, devido a problemas para rodar parte destes exemplos a atividade foi temporáriamente suspensa.
> Além disso, a partir dos feedbacks da última reunião (27/02/2026), foi identificada a necessidade de apresentar melhor os dados obtidos na simulação
> para comparação do simulador FV desenvolvido e o simulador FV do OpenDSS.
> Para tanto, foi desenvolvido um script que utiliza a biblioteca streamlit para exibir os dados de uma forma mais flexível e interativa, que poderá servir
> como base para análises futuras.
