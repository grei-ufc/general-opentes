## 📌 Descrição da Atividade

Compreensão dos dados de entrada da interface e como foram geradas para poder estabelecer uma busca dos elementos dentro do código. Com isso, incluir monitoramentos de corrente e futuramente uma análise das potências. Além disso, houve tentativas de atualização de scripts em Python (utilizando Streamlit e Pandas) para leitura e visualização das corrente de uma simulação extraídos do OpenDSS. A atividade não pôde ser concluída por erro de perda de dados.
